In this short assignment, you'll write a simple brute-force Python script to determine
if one of the potential passwords in the file 'passwords.txt' works for the user 'student' on the host
'springboard.com'

How to run this script:

python bruteforce.py -H <target host> -u <user> -F <password file>

Your task is to complete the missing code in bruteforce.py
